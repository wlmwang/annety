# Introduction
Annety is a multi-threaded reactor pattern network library based on C++11.

# Requires(POSIX & C++11)
    GCC >= 4.8 or Clang >= 3.3
    Linux >= 2.6.28 (for epoll/timerfd/eventfd)
    POSIX(as MACOSX...) (for develop)
    cmake >= 2.8.5

# Build
./build.sh
